使用pyenv+poetry

发表于 2020-06-26

~/open_source » pyenv global pd@pd-mp
system

~/open_source » pyenv version pd@pd-mp
system (set by /Users/pd/.pyenv/version)

~/open_source » pyenv versions pd@pd-mp

  • system (set by /Users/pd/.pyenv/version)
    3.8.2

~/open_source » pyenv global 3.8.2 pd@pd-mp

~/open_source » pyenv global pd@pd-mp
3.8.2

~/open_source » poetry new lingxi pd@pd-mp
Created package lingxi in lingxi

~/open_source » cd lingxi pd@pd-mp

~/open_source/lingxi » python –version pd@pd-mp
Python 3.8.2

~/open_source/lingxi » poetry local 3.8.2 pd@pd-mp
The command “local” is not defined.

~/open_source/lingxi » pyenv local 3.8.2 pd@pd-mp

~/open_source/lingxi » python –version pd@pd-mp
Python 3.8.2

~/open_source/lingxi » vim pyproject.toml pd@pd-mp

~/open_source/lingxi » cat pyproject.toml| grep python pd@pd-mp
python = “^3.8”

添加已有项目需要的依赖进入【tool.poetry.dependencies】(当然也可以通过poetry add package_name进行安装)
[tool.poetry.dependencies]
python = “^3.8”
sanic = “^20.3.0”
Sanic-Cors = “^0.10.0”
xmltodict = “^0.12.0”
pymongo = “^3.10.1”
aioredis = “^1.3.1”
aiohttp = “^3.6.2”
